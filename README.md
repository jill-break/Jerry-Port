### Jerry's Portfolio - Project Overview

* **Full-stack Next.js 16 portfolio application** with integrated frontend and backend
* **Authentication system** using NextAuth.js for secure admin access
* **Admin dashboard** for managing content
* **Project management API** endpoints to create, read, and manage portfolio projects
* **Contact form integration** with API endpoint to handle visitor inquiries
* **SQLite database** managed with Prisma ORM for data persistence and migrations
* **TypeScript** throughout for type safety and better development experience
* **Responsive UI** built with React 19, Tailwind CSS, and Framer Motion animations
* **Modular components** including Hero, About, Portfolio showcase, Contact form, Footer, and Navigation
* **Automated testing** with Jest and React Testing Library for component and API testing
* **Docker containerization** with docker-compose for consistent deployment environments
* **Production-ready deployment** configuration for Render with environment variables and persistent storage
* **ESLint configuration** for code quality and consistency
* **Seed data support** for database initialization and testing
